## Adding Detail Fields

Want to log more than just completion? Add fields in Edit Mode:

| Field Type | Example Use |
|------------|-------------|
| **Number** | Duration (minutes), reps, pages read |
| **Text** | Notes, reflections, what you learned |
| **Select** | Difficulty, location, type of activity |
| **Time** | When you completed it |

When fields exist:
1. Tap **Mark Complete** 
2. Form expands to show fields
3. Fill in details (or skip optional ones)
4. Completion saves with your data

<Tip>
Keep it simpleâ€”one or two optional fields max. The goal is to reduce friction.
</Tip>

## Tips & Best Practices

<AccordionGroup>
  <Accordion title="Start with one habit">
    Master one before adding more. Successful chains build confidence.
  </Accordion>
  
  <Accordion title="Set a specific trigger">

  <Accordion title="Keep the bar low">

  <Accordion title="Use reminders strategically">
    Set reminder 30 minutes before your ideal completion time.
  </Accordion>
  
  <Accordion title="Don't break the chain">

  </Accordion>
